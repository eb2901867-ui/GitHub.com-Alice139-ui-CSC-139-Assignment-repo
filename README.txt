# Student Registration System**Name:** ISREAL ALICE OCHUOLE**Matric Number:** 23/CSC/139**Course Code:** CSC 282**Assignment Title:** Student Registration SystemA modern, single-page style student registration app. Registrations, listing, and deletion happen **without page reloads** via the Fetch API (AJAX).## Features- Smooth AJAX form submission- Instant list refresh- Delete with confirmation (AJAX)- Clean, dark UI- Secure server-side validation- Prepared statements for database safety## Setup1. Create DB and table:   ```sql   CREATE DATABASE IF NOT EXISTS student_db;   USE student_db;   CREATE TABLE IF NOT EXISTS student_records (     id INT AUTO_INCREMENT PRIMARY KEY,     full_name VARCHAR(100) NOT NULL,     email VARCHAR(120) NOT NULL,     department VARCHAR(100) NOT NULL,     matric_number VARCHAR(50) NOT NULL   );   ```2. Update `includes/db.php` with your MySQL username/password if needed.3. Place this folder inside your web server root (e.g., `htdocs` for XAMPP).4. Visit: `http://localhost/testimony_assignment/index.php`## API Endpoints- `GET api/list.php` → `{ success, students: [] }`- `POST api/register.php` (form-data) → `{ success, id }`- `POST api/delete.php` (JSON: `{ id }`)
